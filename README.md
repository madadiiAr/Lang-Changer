# Lang-Changer
## English Version
### Keyboard Language Auto-Corrector

Never paste text in the wrong keyboard layout again.

Have you ever typed or pasted a long piece of text, only to realize your keyboard was in the wrong language? This tool automatically detects and corrects the language of your input based on the physical position of Persian and English keys.
How it works

When you accidentally type Persian text with an English keyboard layout (or vice versa), this application intelligently maps the characters back to their intended language using key position mapping. Just paste your garbled text, and the app will convert it to the correct script.

Features :

  • Real-time text correction between Persian (Farsi) and English keyboard layouts

  • Simple and intuitive GUI built with Tkinter

  • Copy-paste friendly workflow

  • Works with both typing and pasted content

## نسخه فارسی
### تصحیح‌کننده خودکار زبان کیبورد

دیگر هرگز متن را با زبان کیبورد اشتباه وارد نکنید.

تا حالا شده یه متن طولانی رو تایپ یا پیست کنید، بعد ببینید کیبوردتان روی زبان اشتباهی بوده؟ این ابزار به طور خودکار زبان متن شما را بر اساس موقعیت فیزیکی دکمه‌های فارسی و انگلیسی تشخیص داده و تصحیح می‌کند.
نحوه عملکرد

وقتی به اشتباه متن فارسی را با کیبورد انگلیسی (یا برعکس) تایپ می‌کنید، این برنامه با استفاده از نقشه موقعیت کلیدها، کاراکترها را به زبان اصلی‌شان بازگردانی می‌کند. کافیست متن به‌هم‌ریخته خود را پیست کنید، برنامه آن را به خط صحیح تبدیل می‌کند.

ویژگی‌ها :

  • تصحیح لحظه‌ای متن بین چیدمان فارسی و انگلیسی

  • رابط کاربری گرافیکی ساده و روان با Tkinter

  • قابلیت کپی و پیست آسان

  • کارکرد با متن‌های تایپ شده یا پیست شده
